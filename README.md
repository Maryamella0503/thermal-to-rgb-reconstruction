# Thermal-to-RGB Scene Reconstruction

This repository contains the implementation for my dissertation:

**"Thermal-to-RGB Scene Reconstruction Using Depth-Aware ControlNet"**

## Contents
- Baseline pipeline (Stable Diffusion 1.5 + ControlNet)
- Thermal-Aware Depth Normalisation Adapter (TADN)
- Evaluation scripts (edge-based precision, recall, F1)

## Notes
- All experiments were conducted under fixed seeds and identical parameters
- Code is structured to support reproducibility of results presented in the dissertation
